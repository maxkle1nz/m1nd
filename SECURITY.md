# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| 0.1.x   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in m1nd, please report it responsibly.

**Email:** max@cosmophonix.com

**Response timeline:**
- Acknowledge within 48 hours
- Fix for critical issues within 7 days
- Advisory published after fix is available

**Scope:** m1nd-core, m1nd-ingest, m1nd-mcp binaries and their dependencies.

Please do NOT open public GitHub issues for security vulnerabilities.
