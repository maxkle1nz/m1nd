# Internal docs

This folder holds working notes, design explorations, hardening reports, visual review boards, and capability-wave documents that are useful for maintainers but are not part of the primary public docs surface.

## Included here
- landing and README visual notes
- command-center / tray explorations
- Cortex wave design docs
- Surgical hardening and contract notes
- maintainer workflow docs
- other maintainer-facing build/design artifacts

## Public docs that stay outside this folder
- `docs/deployment.md`
- `docs/IDE-INTEGRATIONS.md`
- `docs/wiki/`
- `docs/benchmarks/`
- `docs/AGENT-TASKNOTES.md`

The goal is to keep the public `docs/` surface legible without losing valuable internal history.
